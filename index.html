Chain Reaction - Java Game

Description

A single person project involving the development of a chain reaction java. There are 12 levels in the game and in each level the user will have to explode certain number of balls. The user will firstly click on the game GUI which will make a point for balls to collide. This point will only stay for 3 seconds, in which the other balls have to collide with it. Each ball that collides, will only have a lifetime of 3 seconds, after which the ball will shrink and disappear.

Technologies Used

Java, SQL, Photoshop, SQL Developer, Intellij, Eclipse
